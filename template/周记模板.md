## Fleeting Notes
```dataview
table without id 
	file.link as 日期,
	dateformat(created, "ccc") as 星期, 
	L.text as 总结, 
	L.link as 链接
from "复盘/日报"
where file.mday.weekyear = this.file.mday.weekyear -1
flatten file.lists as L
where
	!L.parent and
	meta(L.section).subpath = "📑 Notes"

```
