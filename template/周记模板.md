
## Fleeting Notes
```dataview
table without id 
	file.link as 日期,
	dateformat(created, "ccc") as 星期, 
	L.text as 闪念笔记, 
	L.link as 链接
from "复盘/日报"
where created.weekyear = this.created.weekyear - 1
flatten file.lists as L
where
	!L.parent and
	meta(L.section).subpath = "## 📆 Journal"

```
